# solve

The **solve** skill translates abstract GCP services into practical "Problem/Solution" stories and diagrams.

## Core Functions

1. **Architectural Story-telling**: Converts abstract service features into "The Problem" and "The GCP Solution" stories.
2. **Case Study Deep-Dive**: Analyzes the official GCP exam case studies and maps them to technical requirements.
3. **Whiteboarding Assistance**: Provides ASCII or Mermaid diagrams to visualize architecture for specific use cases.

## Usage Guide

- **Build Use Case**: Ask the agent to create a use case for "Event-driven microservices" using GKE and Pub/Sub.
- **Analyze Case Study**: Request a breakdown of technical requirements for the "TerramEarth" case study.
