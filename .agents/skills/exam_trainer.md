# Exam Trainer Skill

The **Exam Trainer Skill** acts as a proctor for the mock exams, analyzing performance to guide future study sessions.

## Core Functions

1. **Adaptive Testing**: Adjusts the difficulty and focus of questions based on previous performance.
2. **Performance Analytics**: Identifies "Critical Weak Points" and updates the `tracker/metrics.json`.
3. **Planner Integration**: Automatically suggests adjustments to the `planner/daily/` tasks based on weak score areas.

## Usage Guide

- **Start Simulation**: Conducts a timed 50-minute mock session.
- **Score Report**: Provides a breakdown by exam domain and suggests "Prescription Tasks".
