# distill

The **distill** skill processes raw data into architectural summaries and separates rote flashcards from deep concepts.

## Core Functions

1. **Information Diet**: Converts raw lab notes or documentation snippets into structured markdown in the `content/` folders.
2. **Separation of Concerns**: Specifically separates "Flashcard" rote facts (limits, quotas, ports) from "Deep Concept" architectural summaries.
3. **Cross-Linking**: Identifies common services across certifications and links relevant notes.

## Usage Guide

- **New Note**: Provide the agent with a topic and raw text.
- **Review**: Request a summary of a specific service across PCA/PDE/MLE.
